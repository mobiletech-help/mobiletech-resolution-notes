How to Use MobileTech Resolution Notes

This tool helps you build clean, consistent resolution notes for MobileTech. Nothing is saved. Use Copy All to paste directly into MobileTech when finished.

1) Customer / Lease

  • Optional
  • Applies to all days below it
  • Shown once to keep daily notes clean

Example:

OXY USA – SCR Lease

2) Days & Next Day

  • Each date represents a separate work day
  • Next Day adds a new date AND starts fresh at Crew #1
  • Customer & Lease carry forward automatically

Example:

03/18/26
03/19/26

3) Crews

  • Each day starts with Crew #1 automatically
  • Use Add Crew for additional crews
  • Crews can be expanded or collapsed for easier review
  • If you delete a crew, remaining crews are renumbered automatically

Example:

Crew #1
Crew #2

4) Crew Size & Members

  • Select the crew size
  • The correct number of Crew Member fields appears automatically
  • Names are free‑text

5) Trucks

  • Each crew may have multiple trucks
  • Truck numbers must be exactly 4 digits
  • Letters or shorter numbers are not allowed
  • You can add or remove trucks individually
  • You will be warned if a truck is reused on the same day

Example:

Trucks: 2145, 3057

6) Start / Stop Time

  • Enter the time work started and stopped
  • Used for billing and audit clarity

7) Work Performed

  • Describe what was done that day
  • Include delays, weather, standby, or extra work if applicable

Example:

Installed low pro units and staged pumps.
Standby 1.5 hours due to weather delay.

8) Rentals

  • Add as many rental lines as needed
  • Each line includes:
    • Item / Unit
    • Details (HP, size, type)
    • Qty / Distance
    • Vendor
    • PO
  • Each line can be removed individually

Example:

Low Pro – 600 HP – Qty 1 – Big D Pump – PO 12345

9) Materials

  • Same structure as Rentals
  • Use for materials consumed or sold to customer

Example:

4” Poly – SDR 11 – 1.3 miles – Yard – PO 67890

10) Copy All

  • Copies everything into clean, MobileTech‑ready text
  • Only filled fields appear
  • No extra labels or UI text are copied

Example Output:

OXY USA – SCR Lease

03/18/26
Crew #1
Trucks: 2145, 3057
John, Mike, Carlos
Start: 0700   Stop: 1530
Installed low pro units and staged pumps.

Low Pro – 600 HP – Qty 1 – Big D Pump – PO 12345
