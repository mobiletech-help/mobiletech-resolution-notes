How to Use This Page

This page helps you build clean, consistent MobileTech notes. When finished, click Copy All and paste into MobileTech.

Customer / Lease

Customer and Lease / Location apply to all days unless changed. Both are ALL CAPS and smart‑fill learns what you use.

Trucks

  • Enter all trucks taken to the location (4 digits only).
  • Duplicate trucks on the same day are blocked and identify the crew already assigned.
  • At least one truck is required per crew to Copy All.

Start/Stop

You can add multiple Start/Stop blocks if the crew was called off and returned later. Copy All hard‑blocks if any block is incomplete.

Start/Stop:
• 07:00 AM – 11:30 AM
• 02:15 PM – 06:00 PM

Work Performed

Describe work completed during the day. Include delays, changes, added requests, or issues.

Rentals / Materials

Rentals = returning equipment left on site. Materials = used/sold items not returning. Description fields have smart‑fill suggestions and learn new terms.

Copy All

Copies entered data into clean MobileTech‑ready text: MM/DD/YY date, Crew # – X‑MAN CREW, Trucks, Crew Members, Start/Stop bullets, Work, Rentals, Materials.
