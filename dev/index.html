How to use this page

Customer / Lease are optional and apply to all days.

Next Day adds a new date and resets to Crew #1.

Crews can be expanded or collapsed.

Trucks

  • Exactly 4 digits
  • Multiple trucks per crew
  • Remove individually
  • Warning if reused on same day

Rentals & Materials

Each line includes:

  • Item
  • Details
  • Qty / Distance
  • Vendor
  • PO
