How to Use This Page

This page helps you build clean, consistent MobileTech resolution notes. Nothing is saved to a server. When finished, click Copy All and paste into MobileTech.

Customer / Lease

Enter the Customer and Lease / Location once at the top. These apply to all days below unless you change them.

OXY USA – SCR LEASE

Days & Next Day

Each date represents a separate work day. Next Day adds a new date and starts fresh at Crew #1. Customer and Lease remain.

Crews

Use one crew per working group. You can add additional crews for the same day. Crews can be collapsed to make review easier.

Trucks

  • Enter all trucks taken to the location for the crew.
  • Truck numbers must be exactly 4 digits.
  • Duplicates on the same day are blocked and the conflicting crew is identified.
  • At least one truck is required per crew to Copy All.

Start / Stop Time

Enter accurate start and stop times. These are used for billing and documentation. Times are formatted in AM/PM on Copy All.

Crew Members

  • Type names in ALL CAPS; suggestions appear from previous use.
  • A crew member cannot be assigned to two crews on the same day.

Work Performed

Briefly describe the work completed for the day. Include important context such as delays, scope changes, added requests, and issues.

Installed pumps and staged equipment.
Weather delay approximately 1.5 hours.
Additional work added per customer request.

Rentals (Returning)

Equipment left on location that will be picked up later. Enter each as a row. The Description field has suggestions for common equipment.

Pumps, generators, automation, road crossings, flowmeters, layflat

Materials (Not Returning)

Items used or sold to the customer. The Description field has suggestions for common items.

Liner, geo, fuel, clamps, fittings

Copy All

  • Creates clean, MobileTech‑ready text.
  • Skips empty sections.
  • Includes: Customer – Lease, Date (MM/DD/YY), Crew # – X‑MAN CREW, Trucks, Crew Members, Times (AM/PM), Work Performed, Rentals, Materials.
  • Each rental/material row is a single bullet using the values provided for that row.

Quick Tips

  • Use Add Crew for multiple working groups on the same day.
  • Use Delete Day if a day was added by mistake (a fresh day is added back if none remain).
  • Use Clear All to start over (smart‑fill memory remains on your device).
